# Manual Testing — SauceDemo E-Commerce

## 📋 Project Overview
Manual testing project for [SauceDemo](https://www.saucedemo.com/) e-commerce website covering functional, UI, and security testing.

## 📁 Project Documents
| Document | Description |
|---|---|
| `Test_Plan_SauceDemo.docx` | Test plan with scope, environment, entry/exit criteria |
| `ecommerce_test_cases.xlsx` | 47 structured test cases |
| `Complete_Test_Execution_Report.docx` | Full execution report with PASS/FAIL results |
| `Bug_Report_Ecommerce.docx` | Bug report with severity and steps to reproduce |

## 🧪 Modules Tested
- Login
- Product Listing
- Shopping Cart
- Checkout
- Logout

## 🛠️ Tools Used
- Microsoft Excel — Test cases
- Microsoft Word — Test plan & reports
- SauceDemo website
